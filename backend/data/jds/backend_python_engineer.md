# Backend Python Engineer

We are looking for a Backend Python Engineer to build reliable APIs and services
for high-volume business workflows. The engineer should be comfortable designing
REST APIs, modeling relational data, integrating third-party services, and
debugging production issues.

Key responsibilities:
- Build and maintain backend services using Python, FastAPI, Django, or Flask.
- Design PostgreSQL schemas, write efficient queries, and handle migrations.
- Implement background jobs and async processing with tools such as Celery,
  Redis, queues, or scheduled workers.
- Write automated tests and participate in code reviews.
- Containerize services with Docker and support cloud deployments.

Required skills:
- Strong Python programming and API design experience.
- Practical knowledge of SQL, transactions, indexing, and performance tuning.
- Familiarity with authentication, authorization, logging, and monitoring.
- Experience shipping production backend systems.

Good to have:
- AWS experience.
- Event-driven systems or message queues.
- LLM, RAG, or document-processing experience.
