# DevOps Cloud Engineer

We are hiring a DevOps Cloud Engineer to improve deployment reliability,
infrastructure automation, monitoring, and cloud operations. The engineer should
be comfortable with CI/CD, containers, cloud services, and incident response.

Key responsibilities:
- Build and maintain CI/CD pipelines.
- Manage cloud infrastructure on AWS, Azure, or GCP.
- Containerize applications with Docker and orchestrate workloads with
  Kubernetes or managed container services.
- Automate infrastructure using Terraform, CloudFormation, or similar tools.
- Improve observability with logs, metrics, alerts, and dashboards.
- Support production incidents and perform root-cause analysis.

Required skills:
- Linux, networking, and scripting fundamentals.
- Hands-on experience with CI/CD tools such as GitHub Actions, GitLab CI,
  Jenkins, or Azure DevOps.
- Practical knowledge of Docker and cloud deployment patterns.
- Understanding of security basics, secrets management, and access control.

Good to have:
- Kubernetes administration.
- Site reliability engineering practices.
- Cost optimization and performance tuning experience.
