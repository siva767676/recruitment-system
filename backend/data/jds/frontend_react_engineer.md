# Frontend React Engineer

We are hiring a Frontend React Engineer to build responsive, accessible, and
maintainable user interfaces for web applications. The role requires strong
React fundamentals, state management experience, and attention to performance
and user experience.

Key responsibilities:
- Build reusable UI components using React and TypeScript.
- Manage client-side state using hooks, context, Redux, Zustand, or similar
  libraries.
- Integrate frontend screens with REST or GraphQL APIs.
- Improve page load time, rendering performance, accessibility, and browser
  compatibility.
- Collaborate with designers and backend engineers to ship polished features.

Required skills:
- Strong JavaScript, TypeScript, HTML, and CSS fundamentals.
- Experience with React component design and modern frontend tooling.
- Ability to debug UI issues and reason about rendering behavior.
- Knowledge of testing with tools such as Jest, React Testing Library, or
  Playwright.

Good to have:
- Next.js experience.
- Design system or component library experience.
- Familiarity with CI/CD and frontend observability.
